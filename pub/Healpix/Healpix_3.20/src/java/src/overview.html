HEALPix Java

This is the documentation of the healpix java package.


healpix.core

The core package contains the java implementation of the HEALPix Index. Look in the demo package for example usage. The library now include the 3D rendering package and a tool for explaining HEALPix.

An ant build file is provided in the src package for building the classes and javadocs.
